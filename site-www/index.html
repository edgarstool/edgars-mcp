授權流程

  1. 由核准客戶端或本頁啟動 — ChatGPT、Cursor、Cloudflare Access 等已核准的客戶端會引導您授權；您也可點選上方「Continue with Google」以 Google 帳號驗證身分。
  2. Google 同意畫面 — 您將看到 Google 標準 OAuth 同意畫面，僅請求完成授權所需的最小範圍（openid、email、profile）。
  3. 連結受保護基礎設施 — 授權完成後，身分將用於存取 MCP、Webhook 與其他受 Cloudflare Access 保護的 EDGAR'S 服務。
  4. 隨時可撤銷 — 您可在 Google 帳戶權限頁面 隨時移除本應用程式的存取權限。

本頁面不含任何 client secret 或私密憑證。OAuth client_id 為公開資訊；實際 token 交換僅在受保護的伺服器端完成。

edgarsoauth 是此品牌下的 Google OAuth 授權應用程式，服務於 www.edgars.tools 個人工具生態系。

應用程式用途

edgarsoauth 是 Edgar 個人開發與維運的 OAuth 授權應用程式，用途如下：

  • 讓使用者透過 Google 帳號登入，授權存取 Edgar 的 MCP（Model Context Protocol）伺服器與相關生產力工具
  • 在 ChatGPT、Claude、Cursor 等 AI 客戶端中，安全地連結 Edgar 的本機自動化服務（如檔案管理、Git、Obsidian、Linear 整合）
  • 僅在使用者明確同意後，取得完成授權流程所需的最小 Google API 權限範圍
  • 不會將您的 Google 帳號資料出售、出租或分享給第三方

誰在使用？
